# stock-trading-bot
Objective: to trade stocks during market hours to generate returns using paper trading

# Tech-stack
Python
Alpaca API
Pandas, Numpy, Matplotlib

# Current status
In progress.

# Project Requirements
functional requirements:
- buy/sell positions given algorithmic conditions
- track performance
- chart displays
- provide live market prices and data
- paper trade with fake money (alpaca)
  
  non-functional requirements:
 - executes trades in 1.5 ms
